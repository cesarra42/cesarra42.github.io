Tipos de selectores que entran en examen

  1. Clase (class=””)

    2. Consite en seleccionar varios elemento dentro del documento. De esta manera, todos los elementos que lleven este selector class="algo" tendrán el mismo estilo.

    Ejemplo: Si añado la etiqueta class="importante" a dos elementos cualquiera y les cambio el estilo, solamente cambiara el estilo en esos dos.Los dos primeros tituolos por ejemplo.

  2. Identificador (id=””)

    3. Consiste en seleccionar un único elemneto y ponerle un estilo de manera que pueda ser seleccionado del texto

    Ejemplo: Este parrafo por ejemplo lleva un identificador y por eso solo el tiene este color.

  3. Todos los elementos (*)

    4. La función de este selector es seleccionar todos elementos para poder ponerles el mismo estilo

    Ejemplo:Un ejemplo sería si añadimos la etiqueta al principio del cocumento y añadimos un tipo de letra. De esa forma todo tendrá la misma letra.

  4. Varios elementos ( p, em, h1, h2)

    5. El elemento p, sirve para separar lo que escribimos en parrafos.
    Ejemplo:En este documento por ejemplo, ha sido utilizado continuamnete.

    El elemneto em, sirve para cambiarle el estilo a una o un grupo de palabras para que salgan en cursiva.
    Ejemplo: Sirve para destacar palabras como esta

    Los elementos h1 y h2, sirven para establecer niveles de titulos según su importancia de manera que el h1 es el más importante, el h2 el segundo más importante...y asi continuamente

    Ejemplo:Por ejemplo en este documento hay un h1 que seria el titulo principal y luego varios h2 que son los diferentes titulos de la lista.

    6. Dentro de una etiqueta (ol), que se usa para crear una lista ordenada en este caso, se encuentran otras etiquetas (li), las cuales pueden tener elementos dentro que tengan su propio estilo.
